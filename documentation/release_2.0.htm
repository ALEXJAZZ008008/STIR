Summary of changes in STIR release 2.0 beta (dated 15 May 2009)

Major reworking of the reconstruction hierarchy. See STIR-version2-vs-version1.pdf for more info.

Nearly all work on this update was by Kris Thielemans (Hammersmith Imanet Ltd) and Sanida Mustafovic (Imperial College and Hammersmith Imanet Ltd). Some updates by Charalampos Tsoumpas (Imperial College and Hammersmith Imanet Ltd) have been included as well. In addition, the following has been added:

  • Basic support for SimSET
    contributors: Pablo Aguiar (Univ of Barcelona, Spain), Charalampos Tsoumpas and Kris Thielemans
  • Initial version for parallelisation of OSMAPOSL using MPI
    contributors: Tobias Beisel and Stefan Lietsch (Univ of Paderborn, Germany)
    This work is based on software and documentation from the PARAPET project, but is different in implementation.
  • A contribution by C. Ross Schmidtlein and Assen S. Kirov. Allows to specify 3D boxes as shapes (and hence generate\_image).
